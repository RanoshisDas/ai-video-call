#  AI Companion Video Call & Streaming

## Tech Stack
- Frontend: Nextjs
- Backend: FastAPI

## How to run locally
At first clone the repo using `git clone`

### Frontend
- Step 1: `cd frontend`
- Step 2: `npm install`
- Step 3: `npm run dev`

### Backend
- Step 1: `cd backend`
- Step 2: `pip install -r requirements.txt`
- Step 3: `python main.py`
